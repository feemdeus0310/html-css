Tags em HTML

Em HTML5, podemos inserir os elementos da lista a seguir. É só passar o mouse sobre os termos para descobrir qual a tag deverá ser usada em cada caso

  • Parágrafos
  • Títulos
  • Imagens
  • Abreviações
  • Código Fonte
  • Listas Numeradas
  • Listas com Demarcadores
  • Comentários (não aparecem no seu site, so ficam no código)
  • Estrutura básica do documento e o tipo
  • Cabeçalho de maior relevância,e vai seguindo até o de cabeçalho de menor importância
